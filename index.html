Welcome to Web 101

This mini-course will help you get started with learning HTML and CSS to create webpages and progress to JavaScript and Ember to create web applications.

Prerequisites on Mac OS X

  1. Install homebrew
  2. Install git
  3. Create a GitHub account
  4. Set up git
  5. Install Sublime Text
  6. Install Package Control for Sublime Text
  7. Install Handlebars package for Sublime Text
  8. Install Ember CLI
